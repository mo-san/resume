制作物

  • AIで生成した画像のEXIF情報から、そのプロンプトを読み出すツール。 使用した技術: SolidJS (Reactと同等の機能を持ちながらも軽量なフレームワーク)
  • Live2Dモデルをマスコットキャラクターとしてウェブサイトの片隅に表示するライブラリ。 使用した技術: React, Typescript
  • YouTube 上の楽曲に合わせて 3D モデル(MMDモデル)が動くアプリ。カメラの前で手を振ると手の形を認識して画面上でペンライトが光るという、VRもどきのギミックも。使用した技術: React, Three.js, MediaPipe (姿勢推定ライブラリ)
  • Python製コマンドラインツール。動画サイトの動画をダウンロードするもの。Pythonでは難しい非同期処理を積極的に使用することで、並列処理を可能にしました。 Githubでスターを50弱頂きました。
    関数の一つ一つに詳細な型指定とコメントを添えたので、処理が追いやすくなっています。
    ドキュメントを日英の2言語で用意しており、使い方が一目でわかるように詳細に書きました。
    テストコードも整備し、なおかつ Travis CI でCIを動かしています。
  • VS Code 拡張機能。英数字の全角と半角を切り替える機能があります。ダウンロード数は5,000を超えています。
  • VS Code 拡張機能 2。複数開いたファイルを横に並べた際に、スクロールを同期するもの。諸事情により公開停止。
  • Windows 向けのパッケージマネージャー "Scoop" のリポジトリを運用しています。
